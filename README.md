# hpcc-js

Monorepo for hpcc-js Web Components [documentation](https://gordonsmith.github.io/hpcc-js-wc/)

## Prerequisites

* Node JS >= 16.x
* npm >= 8.x

## Building

```
npm install
npm run build
```

## Website

### Developing

```
npm run -w website start
```

### Deploying to GitHub Pages

```
npm run -w website deploy
```
