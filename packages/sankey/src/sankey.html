Attributes #

link_color #

Link color of the links

Type: "source" | "target" | "source-target" | HTMLColor

  • source: links are colored based on the source node

  • target: links are colored based on the target node

  • source: target - links are colored based on the source and target nodes

  • HTMLColor: links are colored by any valid CSS color string

Default Value: source-target

link_stroke_opacity #

Link opacity

Type: number

Default Value: 0.5

node_align #

Alignment of the nodes

Type: "justify" | "left" | "right"

  • left: nodes are aligned to the left of the diagram

  • right: nodes are aligned to the right of the diagram

  • justify: nodes are evenly distributed in the diagram

Default Value: justify

node_stroke_opacity #

Border opacity of the nodes

Type: number

Default Value: 1

node_stroke_width #

Border width the nodes

Type: number

Default Value: 1

node_stroke #

Border color of the nodes. Value can be any valid CSS color string or "currentColor"

Type: HTMLColor

Default Value: "currentColor"

d3 #

D3 (or D3.js) is a JavaScript library for visualizing data using web standards. D3 helps you bring data to life using SVG, Canvas and HTML. D3 combines powerful visualization and interaction techniques with a data-driven approach to DOM manipulation, giving you the full capabilities of modern browsers and the freedom to design the right visual interface for your data.
